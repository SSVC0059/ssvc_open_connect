# Глоссарий

Краткие определения терминов, которые часто встречаются в документации и в веб-интерфейсе.

| Термин | Значение |
|--------|----------|
| **АС** | Абсолютный спирт — расчётная величина объёма чистого этанола в смеси; в калькуляторе ректификации доли фракций задаются в процентах от АС. |
| **SSVC** | Семейство контроллеров отбора SmartModule (например SSVC0059 V2), с которым OpenConnect обменивается по UART. |
| **OpenConnect** | Прошивка и веб-интерфейс на ESP32, расширяющий управление и телеметрию для SSVC. |
| **OTA / webflash** | Обновление прошивки по сети или через браузер; готовые образы и веб‑установщик — в разделе [Подготовка → Автоматическая установка](preparation/flasher.md). |
| **mDNS** | Локальное имя вида `http://ssvcopenconnect.local` для доступа к веб-интерфейсу после подключения к Wi‑Fi (если поддерживается сетью и ОС). |
