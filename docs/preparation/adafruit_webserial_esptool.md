# Прошивка через Adafruit WebSerial ESPTool


В случае когда по различным причинам прошивка через встроенный прошивальщик SSVC Open Connect не срабатывает, можно воспользоваться сервисом
Adafruit WebSerial ESPTool

Для этого:

1. Скачиваем архив с прошивкой на странице <a href="https://github.com/SSVC0059/ssvc_open_connect/releases" target="_blank">релизов</a>  проекта  

2. Переходим по <a href="https://adafruit.github.io/Adafruit_WebSerial_ESPTool/" target="_blank">ссылке</a> на сайт проекта

3. Для прошивки контроллера наиболее верным будет подключение через разъем USB&OTG. При этом необходимо зажать кнопку BOOT перед подключением контроллера к компьютеру и перевести его в режим загрузки бутлоадера

Затем выставляем битрейд 115200 и нажать кнопку Connect

![](media/afruil_1.png)

4. При первоначальной загрузке прошивки, а так же во всех странных проблемах следует сделать очистку, нажатием на кнопку *ERASE*

При успешной очистки лог будет примерно такой

![](media/afruil_2.png)

5. Выбираем файлы прошивки и указываем им верные адреса смещения

```
0x00000 - bootloader.bin 
0x8000 – partitions.bin
0x10000  - SSVC-Open-Connect_esp32-s3-devkitc-1-16m_0-1-8.bin
```

затем нажимаем кнопку `PROGRAM`

![](media/afruil_3.png)


Для обновления прошивки достаточно прошить только один файл прошивки по адресу

```
0x10000  - SSVC-Open-Connect_esp32-s3-devkitc-1-16m_0-1-8.bin
```
