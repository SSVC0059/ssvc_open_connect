# Скачивание релиза SSVC Open Connect

Актуальные релизы  всегда хранятся на странице релизов проекта в [github](https://github.com/SSVC0059/ssvc_open_connect/releases)


## Виды релизов

Релизы разделяются на несколько видов - стабильный и тестовый

 - `Стабильный (Latest)` - предназначен для постоянной работы. В этом релизе не будет вноситься новый функционал, но до выхода нового стабильного релиза будут исправляться ошибки. Для первого запуска стоит использовать именно его, так как по большей части все проблемы на нем решены и проверен большим количеством участников

 - `Предварительный (Pre-release)` - Релиз который потенциально может стать стабильным и отдается на тестирование опытным пользователям. Может включать новый функционал, а так же и новые проблемы. Не рекомендуется к установке новичкам.

 - `Ночная сборка` - сборка прошивки с полследними изменениями. Особо не протестированна и с большой вероятностью может содержать проблемы. Только для опытных пользователей и тех, кто знает что он делает. Большая вероятность окирпичить ESP32 и потребуется откат старой прошивки через провод

## Как скачать?

!!! info inline end
    Для контроллера N16R8 - с префиксом 16m

    для контроллера N8R8 - с префиксом 8m


Для первоначальной прошивки скачиваете полный архив, соответствующий вашему контроллеру. 

!!! warning
    Прошивка контроллера не верной прошивкой приведет к окирпичиванию контроллера. Придется перепрошивать заново, но уже верной.
    Будьте внимательны

### *Ночная сборка*

Потребуется учетная запись на github в противном случае ссылка активна не будет

Переходим на страницу релизов и выбираем нужную сборку. Она должна быть успешной (зеленой)

![](media/night_build_1.png)

Выбираем нужный тип контроллера и разворачиваем задачу `Uploading PR artifacts`

Скачиваем по ссылке архив с прошивкой

![](media/night_build_2.png)
