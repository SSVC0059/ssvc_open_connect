# Датчик температуры DS18B20


Open Connect поддерживает произвольное подключение дополнительных цифровых датчиков температур `DS18B20`
Подключение осуществляется через GPIO2

Питание датчиков осуществляется напряжением 5V от внешнего блока питания. GND общее с контроллером

Для стабильной работы нужна подтяжка резистором на 4.7кОм. В продаже есть готовые модуля с резистором.
