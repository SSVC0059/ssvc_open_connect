# Подключение адресного светодиода

Open Connect для удобства может отображать статус подключения с помощью определенного цвета на адресном светодиоде.
Поддерживается как встроенный светодиод, так и возможно подключение внешнего

## Встроенный светодиод

### Статусы встроенного светодиода

<span style="color: #FF0000;"> КРАСНЫЙ </span> - в офлайн режиме

<span style="color: #42AAFF;"> ГОЛУБОЙ </span> - режим точки доступа. Контроллер ожидает подключение клиентов

<span style="color: #0000FF;"> СИНИЙ </span> - режим точки доступа. К контроллеру есть активные подключения

<span style="color: #FFFF00;"> ЖЕЛТЫЙ </span> - контроллер подключен к роутеру и ожидает подключение клиентов

<span style="color: #00FF00;"> ЗЕЛЕНЫЙ </span> - контроллер подключен к роутеру и есть активные подключения

Для включения встроенного светодиода необходимо запаять перемычку питания на плате рядом со светодиодом

## Внешний светодиод

Для подключения подойдут светодиоды WS2812B или APA106. Но со внешними светодиодами могут быть несколько важных моментов

1. цветовая схема у них может быть разная, то есть статусы контроллера могут отображаться другим цветом. Это специфика самого светодиода и не влияет на работу. Нужно будет просто привыкнуть

2. Перед подключением обязательно разберитесь с питанием светодиода. Переплюсовка может сжечь светодиод

Входящий пин светодиода - подключать в `GPIO48` контроллера

!!! warning "ВНИМАНИЕ!"

    Некоторые светодиоды могт прийти вообще со странной схемотехникой, например APA106 мне пришел с перепутанными пинами VCC и GND. Это привело к тому, что несколько светодиодов спалил, пока разобрался.

По этому, прежде чем подключать убедитесь, что распиновка верная. В моем случает это было вот так. По этому спросите у продавца - это важно.
